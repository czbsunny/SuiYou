# 机构选择页面重构计划

## 一、需求分析

参考 web 端的机构选择页面 `d:\code\SuiYou\web\src\pages\assets\institution-selector.vue`，改进 app 端的搜索机构页面。

### web 端特点：
1. 搜索框使用图片图标（search.png），带有清除按钮
2. Tab 切换使用下划线指示器而非背景高亮
3. 列表按字母分组展示
4. 右侧有索引条可以快速定位

### app 端当前状态：
1. 搜索框使用 emoji 🔍
2. Tab 使用圆角背景高亮
3. 简单的列表展示
4. 没有索引条

## 二、修改方案

### 2.1 拷贝静态资源
- 从 web 端拷贝 `search.png` 图标到 app 端

### 2.2 更新搜索框
- 使用图片图标替换 emoji
- 添加清除按钮功能

### 2.3 更新 Tab 样式
- 使用下划线指示器替代背景高亮
- 参考 web 端的 Tab 样式

### 2.4 更新列表布局
- 添加按字母分组功能（可选，根据复杂度评估）

3. **功能完整性**：清除按钮需要绑定事件处理
